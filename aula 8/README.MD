
Este programa serve para **organizar arquivos automaticamente** dentro de uma pasta.

Você só precisa:
1. Escolher uma pasta  
2. Selecionar como quer organizar (por tipo, por data ou ambos)  
3. Clicar em **Organizar**  

Pronto! O programa cria pastas e move os arquivos sozinho.

---

## ⚙️ Como funciona (bem simples)

- Ele lê todos os arquivos da pasta  
- Identifica o tipo (imagem, documento, vídeo, etc.)  
- Verifica a data do arquivo  
- Cria pastas automaticamente  
- Move os arquivos para o lugar certo  
- Se já existir um arquivo com o mesmo nome, ele cria outro nome (não sobrescreve)

---

## 🤖 Como os prompts ajudaram

Os prompts foram usados para **evoluir o projeto passo a passo**:

- Primeiro: criar um script básico  
- Depois: organizar por tipo  
- Depois: evitar arquivos duplicados  
- Depois: permitir escolher a pasta  
- Depois: criar uma interface com botões  
- Por fim: adicionar opções (tipo, data ou ambos)

Ou seja, os prompts ajudaram a transformar uma ideia simples em um **programa completo com interface gráfica**.
